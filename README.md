# Carpool Bot - Поиск попутчиков Уфа-Акъяр

Telegram бот для поиска попутчиков между Уфой и Акъяром.

## Возможности
- Регистрация водителей и пассажиров
- Создание заказов
- Бронирование мест
- Система рейтингов
- Чат с поддержкой

## Установка
1. `git clone https://github.com/artura44-prog/my_bot`
2. `cp .env.example .env` и заполните токены
3. `docker-compose up -d`

## Команды
- /start - главное меню
- /admin - панель администратора
